🛡️ SheShield – Smart Safety Ring App
🚨 Overview
SheShield is a smart emergency safety system designed for real-life panic situations where accessing a phone is difficult or impossible.

In critical moments, people may panic and lose the ability to operate their phone. SheShield solves this by combining a wearable smart ring with an Android application that can trigger emergency actions instantly and automatically.

👉 Works even without internet — safety at your fingertips.

💡 Concept
The system uses a smart safety ring with a recessed button.

👉 Pressing the button 3 times activates an SOS trigger in the mobile app.

To prevent accidental activation, a 10-second safety timer allows the user to cancel the alert.

🔗 How It Works
User presses the smart ring button 3 times

App receives the trigger signal

A 10-second timer starts

User can cancel if accidental
If not cancelled, SOS activates:

📍 Location fetched
📩 SMS sent to emergency contacts
📞 Call placed to police
🔊 Siren activated
Nearby safety locations are shown
